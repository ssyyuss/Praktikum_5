Kesimpulan skenario 1 : Fungsi flexbox untuk mengatur wadah_flex dan elemen di dalamnya menjadi sejajar ke kanan.

Kesimpulan skenario 2 : Kita bisa menggunakan selector .wadah_flex > div untuk mengatur style pada setiap div anak dari class wadah_flex.

Kesimpulan skenario 3 : Fungsi flex-direction untuk mengatur arah urutan pada elemen-elemen yang ada dalam class wadah_flex. Column untuk vertikal, row untuk horizontal, atau bisa juga untuk lawan arahnya dengan menggunakan reverse.

Kesimpulan skenario 4 : Semua items di dalam wadah_flex akan berada pada satu baris meskipun ukurannya sudah tidak cukup, wrap memungkinkan untuk memindahkan items ke baris dibawahnya.

Kesimpulan skenario 5 : Dengan nilai row wrap, elemen-elemen flex akan ditempatkan dalam baris secara default. Namun, jika ruang yang tersedia dalam wadah_flex tidak mencukupi untuk menampung semua elemen dalam satu baris, elemen-elemen tersebut akan dipindah ke baris berikutnya.

Kesimpulan skenario 6 : justify-content digunakan untuk mengatur penempatan elemen-elemen flex dalam sumbu horizontal dalam wadah_flex.
